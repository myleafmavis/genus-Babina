Explanation of abbreviations in this data sets

'Music' stands for Babina_daunchina (Music forg), abbreviated as 'BD';
'HN' stands for Babina_hainanensis, abbreviated as 'BH';
'SQ' stands for Babina_okinavana, abbreviated as 'BO';
'LQ' stands for Babina_lini, abbreviated as 'BL';
